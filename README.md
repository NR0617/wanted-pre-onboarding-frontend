### 프로젝트 소개

-   지원자의 성명
    오나래

-   프로젝트의 실행 방법

    1. git clone https://github.com/NR0617/wanted-pre-onboarding-frontend.git
    2. npm install
    3. .env 파일 생성 후 REACT_APP_SERVER_URL에 배포된 서버 경로를 연결
       e.g. REACT_APP_SERVER_URL = "https://~"
    4. npm start
